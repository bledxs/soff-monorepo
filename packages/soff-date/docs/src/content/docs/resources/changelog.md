# Changelog

All notable changes to Soff Date are documented here.

## [0.2.0] - 2024

### Added
- Business days logic
- Colombia holiday offsets fixed
- Improved test coverage

### Changed
- Updated README with better examples

## [0.1.0] - Initial Release

### Added
- Core holiday calculation engine
- Support for Colombia, USA, Mexico, Argentina, Brasil
- i18n support (English, Spanish, Portuguese)
- Holiday rules: fixed, nthWeekday, easterRelative, custom
- Shift rules: emiliani, observedUS, nextMonday, nearestMonday
- Business day calculations
- TypeScript support
- Zero dependencies
